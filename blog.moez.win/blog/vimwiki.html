常用语法

  • %nohtml 加入这个标记不会生成html
  • ---- 4个'-'会产生分割线
  • %% 开头的为注释,不会生成html代码

字体

  *bold text*
  _italic text_
  ~~strikeout text~~
  `code (no syntax) text`
  super^script^
  sub,,script,,
示例

bold text italic text strikeout text code (no syntax) text superscript subscript

编辑时的按键

  1. 在词条上敲回车键，可以编辑这个词条
  2. 使用 Tab 键，可以跳到下一个维基词条或链接，使用 Shift-Tab 跳到上一个
  3. 在标题上点击 - 和 =，可以提升和降低标题层级
